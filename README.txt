# Steam Game Downloader

A Gradio-based interface for managing Steam game downloads via SteamCMD.

## Features

- SteamCMD auto-installation
- Secure credential management
- Real-time download progress
- Public link generation
- Cross-platform support

## Quick Start

### Local Installation
```bash
git clone https://github.com/your-repo/steam-downloader.git
cd steam-downloader
pip install -r requirements.txt
python app.py
```
